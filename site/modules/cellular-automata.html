Cellular Automata Grid

A grid of cells, each one a small state machine that only ever looks at its eight neighbors. Implement int nextState(int state, int[] neighbors) { ... } — called once per cell, every generation — and the whole grid evolves from your rule alone. Conway's Game of Life is one rule out of many: load WireWorld, Brian's Brain, or a two-color "Immigration" variant of Life from the presets, stamp in a glider or a Gosper glider gun, or write a rule nobody's seen before.

Objective

One local rule, one global pattern

Every cell only ever knows its own state and its eight neighbors' states — there's no global logic anywhere. Start with Conway's Life and stamp in a Glider or the Gosper Glider Gun to see motion emerge from nothing but birth/survival counts. Then switch to WireWorld (a 4-state rule used to build real logic circuits out of cells) or Brian's Brain (an oscillating 3-state rule with no stable still lifes at all), and notice that both run through the exact same grid and playback controls — only nextState() changed. Finish by writing your own multi-color rule: the Immigration preset is a two-color version of Life where a newborn cell inherits the majority color of the three parents that created it.

Reference

nextState(state, neighbors) API

state
The current cell's state, an integer from 0 (background) up to however many colors the active rule uses.
neighbors
An 8-element array of the Moore neighborhood's states, in order N, NE, E, SE, S, SW, W, NW.
count(neighbors, s)
Helper available inside your rule — returns how many of the 8 neighbors are in state s.
return value
The state this cell becomes next generation. Every cell in the grid is evaluated against the same old generation, so order never matters.
Run Rule
Recompiles nextState(...) and (if checked) applies it going forward as the grid steps.
wrap edges
When on, the grid is toroidal — the right edge's neighbor is the left edge, and the bottom's is the top.
palette
Swatches match the active rule's state count. Pick one, then click/drag on the grid to paint cells with that state.
Famous Patterns
Stamps a known still life, oscillator, spaceship, or gun onto the center of the grid (clears the board first).
