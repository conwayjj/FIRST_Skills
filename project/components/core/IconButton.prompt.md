**IconButton** — square icon-only control for social links, nav toggles, close buttons. Always pass a `label` for accessibility; put the SVG glyph as children (Lucide recommended).

```jsx
<IconButton variant="solid" label="Instagram"><InstagramSvg/></IconButton>
<IconButton variant="ghost" label="Menu"><MenuSvg/></IconButton>
```

Variants: `solid` `outline` `ghost` `dark`. Sizes: `sm` (32) `md` (40) `lg` (48).
