**Button** — the brand's primary call-to-action; uppercase squared label on Team Blue. Use for "Learn More", "Donate", "Join a Team" style actions.

```jsx
<Button variant="primary" size="lg">Learn More About Our 2026 Season</Button>
<Button variant="secondary">Click Here to Learn More</Button>
<Button as="a" href="/donate" variant="dark">Donate</Button>
```

Variants: `primary` (blue fill), `secondary` (blue outline), `ghost` (text only), `dark` (near-black fill). Sizes: `sm` `md` `lg`. Pass `iconLeft`/`iconRight` for glyphs, `fullWidth` to stretch.
