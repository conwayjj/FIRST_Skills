**StatCard** — a big display figure with a supporting label, for stat strips like "Facts About FIRST".

```jsx
<StatCard value="785K+" label="students participated in the 2023-2024 season" />
<StatCard value="81%" label="of FIRST® Alumni declare majors in STEM" onDark />
```

Props: `value`, `label`, `align` (`left`/`center`), `onDark` for inverted color.
